# DQ STUDIO - Modulo inicial obra v1

Alcance inicial:
- Clientes
- Obras / expedientes
- Control de ejecucion
- Seguridad y Salud
- Actas/documentos imprimibles
- Gestion de cobros

Instalacion GitHub Pages:
1. Subir el contenido de la carpeta `app` a la raiz del repositorio.
2. Activar GitHub Pages en Settings > Pages > main > /(root).
3. Abrir la URL publicada.
4. En iPhone/iPad: Safari > Compartir > Anadir a pantalla de inicio.

Datos:
- Esta version guarda datos en localStorage del navegador.
- Sirve para validar el flujo antes de conectarlo a Supabase.
