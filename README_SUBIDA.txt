DQ STUDIO MODULO OBRA - VERSION DEFINITIVA OPERATIVA

INSTRUCCIONES:
1. Descomprime este ZIP.
2. En GitHub entra en el repositorio DQ_STUDIO_MODULO_OBRA_v1.
3. Abre la carpeta app.
4. Pulsa Add file > Upload files.
5. Arrastra TODO el contenido de esta carpeta descomprimida: index.html, manifest.json, sw.js y assets.
6. Confirma con Commit changes.
7. Espera 1-3 minutos y abre la URL de GitHub Pages.

NOTA:
Los datos se guardan en el navegador mediante localStorage. Usa Exportar copia para hacer copias JSON.
